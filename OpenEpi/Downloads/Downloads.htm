For All Operating Systems With Browsers

  • Download one of the files on the right. Both contain the OpenEpi files, but the MSI makes installation easier for Windows users.

  • Unzip to a folder on your hard disk (e.g., C:/OpenEpi/). Unzipping programs differ; be sure that yours will create the /OpenEpi/ folder.

  • To use the programs, run index.htm in a JavaScript-enabled browser, or click on the OpenEpi folder.

  • Output can be saved with the SAVE AS command in the browser.

For Microsoft Windows(R) and Epi Info(TM) Users

  • OpenEpi.MSI is a Microsoft Installation file. Save it to your hard disk, and then run it from the download dialog or by double-clicking its icon. It will automatically place several shortcut icons on the desktop to make running OpenEpi easier. Feel free to delete any that you would rather not have.

  • The OpenEpi icon runs OpenEpi in your default browser just as on the Net. The OpenEpiSave icon runs a file called OpenEpiSave.HTA in Internet Explorer to save output pages automatically in /RESULTS/.

  • The OELink icon allows you to edit the main Epi Info menu to insert links to OpenEpi. You will probably only run this program once, after which you may want to delete its icon.
